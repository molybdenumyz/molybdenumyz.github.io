Eclipse上配置python编译环境

课设想用python写图形界面，也就拾起了上学期入了点门的python，电脑重装后新系统一片空白，那正好从头配置下python编译环境

先说下Windows环境

python全局环境

走一波官网
https://www.python.org/downloads/
找到对应的版本，你的操作系统位数下载就好，选择默认路径安装后，添加系统变量这些就不累赘了

cmd 进入命令行后 输入 python

看到图示的样子就没问题了

看了下丑陋的shell 没有代码提示没有缩进判断等（对于python 缩进是要命的）简直是反人类，在linux上用phpstrom写php这让我直接想到了
JetBrains家族下的pycharm,然后对于我这种没有ssd的渣渣电脑 那个开启速度让我瞬间不想写代码了🙃

突然想起u盘里还有图灵杯配置环境时留下来的eclipse，打开速度虽然也不咋地，不过比pycharm好多了，于是决定换eclipse，写java的同学肯定知道这个编译器，然而怎么让它成为你得心应手的python编译器呢，这就是重点了

同样官网走一波
https://eclipse.org/downloads/
下载eclipse for java 安装就不累赘了，然后我们知道没有java环境eclipse是打不开的
这时候就要去java官网
http://www.oracle.com/technetwork/java/javase/downloads/index-jsp-138363.html
获取java了,选择JRE，然后找到对应系统的包就可以

在C:\Program Files目录找到java文件夹里的bin 复制它的绝对路径
回到eclipse的安装目录下 找到 eclipse.ini
用文本编译器打开在openFile下加入路径

保存之后就可以使用eclipse了

进入界面之后点选Help->install New Software

点击add按钮之后按图中操作

eclipse会自动解析出能用的拓展,选择第一个，然后finish,他会自动下载一路确定就可以了

网速慢的同学可能要等很长时间哟

安装完毕后会提示重启eclipse，重启后添加新的项目 会有pydev选项这就表示pydev添加成功


按图中选择就可以创建一个python项目了

进入编译器配置后 点NEWS 将python的安装目录输入 并取名就好了

一切就绪后写一个helloworld程序检测下吧~
底部控制台会有对应的结果显示出来
以后java python 编程就两不误啦~
