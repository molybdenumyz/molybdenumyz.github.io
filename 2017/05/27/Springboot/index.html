Springboot +mybatis + Tomcat 部署

部署从理论上来说十分的简单，但是运维过程还是有点艰辛的，大致说一下我的部署过程

springboot打包成war

  1. 修改打包方式和命名

  1. 去除内置的Tomcat

  2. 添加Tomcat servlet-api

  3. 解决某些java包找不到的问题 （path Unix用：分割，Windows用；）

  4. 添加mevan打war包插件 这里也可以指定包名



    @SpringBootApplication
    @MapperScan("com.neuqer.voter.mapper")
    @EnableAutoConfiguration    
    public class VoterApplication extends SpringBootServletInitializer {

        public static void main(String[] args){
            SpringApplication.run(VoterApplication.class,args);
        }
        @Override
        protected SpringApplicationBuilder configure(SpringApplicationBuilder builder) {
            return builder.sources(VoterApplication.class);
        }
    }
